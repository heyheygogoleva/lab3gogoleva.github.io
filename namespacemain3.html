def main3.hash_table1 (   arr) 
Функция для создания хэш-таблицы для первой хэш-функции.

Позволяет создать хэш-таблицу на основании списка, каждый элемент которого относится к классу Table. Таблица реализована с помощью словаря, ключи которого - значение первой хэ-функции каждого элемента, значения - сам элемент. В качестве способов избежания колллизий выбран метод цепочек, то есть на один ключ приходится список значений из объектов класса Table, хэш-функции которых совпадают (хэшированное значение с помощью первой хэш-функции).

Аргументы: список из элементов класса Table (list[Table])

Возвращает: хэш-таблицу (dict {Table.hash1 : list[Table]}), количество коллизий (int)
